## Рынок US LIMS — размер и расчёт

| Метрика | Значение |
|---------|----------|
| US LIMS market 2024 | $735M |
| Прогноз 2030 | $1.30B |
| CAGR | ~10% |
| Сравнение: лабораторная информатика в целом | 3.6% CAGR |

Опережающий рост — за счёт cloud/SaaS, где стоит Vivica.

### TAM → SAM → SOM

| Уровень | Размер | Что это |
|---------|--------|---------|
| **Все CLIA-лабы США** | ~300,000 | Маркетинговая абстракция, не рынок |
| **TAM** (POL / PSC / Reference, без hospital и waiver) | ~25,000 | LIMS-релевантный рынок |
| **SAM** (Compliance + Accreditation cert) | **9,193** | Реальный охватываемый рынок (CMS Q1 2026) |
| **SOM** (5 лет, 1-3% от SAM при качественном GTM) | 150-600 контрактов | Целевой |

**Ключевое**: 72% POL работают по waiver-сертификату — точечные тесты, LIMS не нужен. Это критично для приоритизации: «300K labs» — не наш рынок, реальный охватываемый меньше на порядок.

## Воронка — Reference segment

```
CMS CLIA Q1 2026 → Reference bucket                 1,849 CLIA-записей
  → Domain enrichment (Clay + Exa)                  1,809 записей с доменами
  → Уникальных компаний (dedup по домену)           1,019
  → Apollo hit (есть люди)                            383 доменов (37.6%)
  → Контактов найдено                               1,252
  → Apollo Reveal (email)                            1,033 (82.5%)
  → FindyMail SMTP double-verify                       878 (85%)
  → Dedup по email                                     344 уникальных контактов ✅
  → Уникальных компаний с контактами                   270
  → Dark (без контактов)                               749 компаний
```

**Стоимость**: ~$22.70. $0.034 за верифицированный контакт.

**Ограничение Apollo**: 14.4% bounce при SMTP-check → double-verify обязателен всегда.
**Структурное**: Apollo пропускает ~62% reference-лаб (нет LinkedIn). Для dark labs нужен LinkedIn Sales Nav или Clay waterfall.
